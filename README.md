# Week 3 Day 1 - Machine Learning Fundamentals

## Project Overview

This project demonstrates the complete Machine Learning workflow using Scikit-learn and Linear Regression. The project predicts student marks based on study hours, attendance, and assignments. It includes data exploration, model training, evaluation, predictions, and data visualization.

---

# Objectives

The main objectives of this project are:

* Understand the Machine Learning workflow
* Identify features and labels
* Perform train-test split
* Train a Linear Regression model
* Predict marks using new data
* Evaluate model performance
* Create visualizations using Plotly

---

# Dataset Information

The dataset contains the following columns:

| Column Name | Description                     |
| ----------- | ------------------------------- |
| Study_Hours | Number of hours studied         |
| Attendance  | Attendance percentage           |
| Assignments | Number of assignments completed |
| Marks       | Student marks                   |

# Train-Test Split

The dataset is divided into:

* 80% Training Data
* 20% Testing Data

Training data is used to train the model, while testing data is used to evaluate model performance.

---

# Model Used

## Linear Regression

Linear Regression is used to predict student marks based on input features. It finds the relationship between independent variables and dependent variables.

---

# Model Evaluation

The model is evaluated using:

* Mean Absolute Error (MAE)
* R² Score

A low MAE and high R² Score indicate better model performance.
